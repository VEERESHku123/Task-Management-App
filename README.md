# Task-Management-App
A simple and efficient Task Management App built using Spring Boot for the backend and React for the frontend. It helps users add ,delete and manage tasks efficiently.
##Tech Stack
### Backend (Spring Boot)
Spring Boot
Spring Data JPA
Hibernate
H2
Maven
### Frontend (React)
React.js
Axios
React Router
CSS
